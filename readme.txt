=============
ActivityGraph
=============

ActivityGraph is an application for visualizing developer workflow. You can find more information about ActivityGraph at http://cse.unl.edu:8080/projects/ActivityGraph

Running ActivityGraph
----------------------
to run ActivityGraph, simply type::

 java -jar ActivityGraph.jar

Try opening the XML file data/data.xml to try out the visualization (using the File menu).

You can also try using our Subversion repository as a data source at http://cse.unl.edu:8080/svn/ActivityGraph.
